Odigos Automatic OpenTelemetry Instrumentation Agent for Virtual Machines.

## Introduction

- Instrument Systemd services running on a Linux VM.
- No code changes - Automatic OpenTelemetry instrumentation with eBPF.
- Instance distributed traces - Odigos takes care of the heavy lifting and provides a seamless experience.
